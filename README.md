# Adversarial Image Concept (No ML)

This project demonstrates the **mathematical concept of adversarial perturbations in image processing** without using machine learning models.

The goal is to show how **small, carefully crafted changes to an image can significantly alter its interpretation**, highlighting the vulnerability of image-based systems.

## Features
- Demonstrates adversarial perturbation mathematically
- Works without any machine learning framework
- Simple Python implementation
- Helps understand the core idea behind adversarial attacks

## Technologies Used
- Python
- NumPy
- Image Processing Concepts

## How It Works
The program slightly modifies pixel values of an image to create **adversarial perturbations**.  
Even though the image appears almost unchanged to the human eye, the internal representation can be significantly affected.

## Applications
- Understanding adversarial attacks
- Image security research
- Educational demonstrations of adversarial concepts
